# glue_sticks_organizer
creating an organizer for keeping glue sticks
# **:wrench: Создание настенного органайзера для хранения клеевых стержней**
## Обзор решения
Универсальным и наиболее удобным решением для хранения большого количества инструментов является использование стенки типа skadis. Такая конструкция обеспечивает эргономичность, экономию пространства и удобство в использовании. Итак, наше изделие должно встраиваться в концепцию данной стены, а также соответствовать главным критериям:
- мобильность (возможность легко перемещать органайзер);
- компактность;
- удобство использования;
- прочность и долговечность;
- экономность в использовании материала.
Давайте спроектируем органайзер, сочетающий использование разных технологий производства.

## Дизайн модели и производство
1. Поиск аналогов, выбор технологии производства

Поиск аналогов привёл к выводу, что удобнее всего хранить стержни в стакане цилиндрической формы. Стакан с крепежом, произведенный одной деталью, неоправданно много затратит ресурсов, печать такой детали требует много времени ( 4 часа) и материала. Соответственно, намного эффективнее использовать составные части, а вместе с ними и разные технологии
